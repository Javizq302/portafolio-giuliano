# Videos

Coloca aquí los videos del portafolio.

**Formatos soportados:** MP4, WebM, MOV

**Recomendaciones:**
- Optimiza los videos para web (compresión adecuada)
- Considera usar diferentes calidades (HD, Full HD) según necesidad
- Para videos grandes, considera usar un servicio de hosting de video (YouTube, Vimeo, etc.)
