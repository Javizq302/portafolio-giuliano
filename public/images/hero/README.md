# Imágenes del Hero

Coloca aquí la imagen principal de Giuliano Medina para el Hero.

**Nombre del archivo:** `giuliano.jpg` (o `.png`, `.webp`, etc.)

**Ubicación:** `/public/images/hero/giuliano.jpg`

**Recomendaciones:**
- Formato: JPG, PNG o WebP
- Tamaño recomendado: Al menos 800x800px para buena calidad
- La imagen se mostrará en formato circular
- Optimiza la imagen antes de subirla para mejor rendimiento
